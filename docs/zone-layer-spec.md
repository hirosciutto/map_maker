# 地理圏レイヤー(レイヤー2)仕様

- 版: v0.2(2026-07-12。14圏の動物地理区へ更新 = blockland design v2.05)
- 目的: Blockland の生息域仕様「**地理圏 × バイオーム**」(blockland/docs/design.md §18.4/§18.5.1 v2.04)のための `zone_map.json` を、本ツールで塗って出力できるようにする。
- 発注元: blockland design TBD #65。

## 1. 概念

既存のバイオームマップ(レイヤー1)の**上に重なる第2のグリッド**。フルのレイヤーシステムは不要で、必要なのは次の3点だけ:

1. **下敷き表示**: レイヤー2 編集中は、レイヤー1(バイオーム)を**淡色(減光)で下敷き表示**する。大陸の形をなぞって塗るための参照。
2. **海マスク**: レイヤー1 が水域バイオーム(`OCN`/`SHF`/`DPO`/`TRN`)のセルは**暗くマスクされ、塗れない**(既存の「パレット中央チェックによるバイオームマスク」機構の流用を想定)。
3. **地理圏パレット**: レイヤー2 編集中はパレットが**地理圏14色**に切り替わる。

## 2. 地理圏コード(14圏・固定3文字)

動物地理区(Holt 2013 / WWF)ベース。旧北区は動物のルーツ解像度のため西旧北 `WPA`/中央旧北 `CPA`/中日 `SIN` に3分割。

| code | 和名 | 暫定色 | code | 和名 | 暫定色 |
|:---:|---|:---:|:---:|---|:---:|
| `NEA` | 北極圏以外の北アメリカ | `#e6194b` | `PAN` | 中米・カリブ | `#fabed4` |
| `NEO` | 南アメリカ | `#ffe119` | `AFR` | サハラ以南アフリカ | `#9a6324` |
| `MDG` | マダガスカル | `#808000` | `SAH` | サハラ・アラビア | `#f58231` |
| `WPA` | 西旧北区(欧州・地中海) | `#3cb44b` | `CPA` | 中央旧北区(中央アジア・シベリア) | `#469990` |
| `SIN` | 東アジア(中国東部・朝鮮・日本) | `#4363d8` | `IND` | 南・東南アジア | `#42d4f4` |
| `AUS` | オーストラリア・ニューギニア | `#f032e6` | `OCE` | 太平洋諸島・NZ | `#dcbeff` |
| `ARC` | 北極圏 | `#aaaaaa` | `ANT` | 南極圏 | `#eeeeee` |

- 色は識別用の暫定(隣接圏が判別できれば良い。バイオームパレットのような最近傍色検証は不要)。**14色は categorical パレットで十分に区別可能**(paintability 上限 ~20色の範囲内)。
- コード表はデータ駆動にし、追加・改名できること。

## 3. 塗りのルール

- **粒度は粗くて良い**。バイオームのような1pxの精密さは不要(目的はゲーム上の生息分けであり、境界の厳密な地理ではない)。
- **全陸セル(水域バイオーム以外)が塗られていること**が完成条件。未塗り陸セルは書き出し時に**検証エラー**にする。
- 島は最寄り圏に含める(例: マダガスカル=`AFR`、日本=`EAS`、ニュージーランド=`OCE`、グリーンランド=`ARC`)。
- 北極圏(`ARC`)・南極圏(`ANT`)は緯度帯ではなく「極地の陸・氷床」を塗る(グリーンランド、シベリア最北縁、南極大陸)。

## 4. 出力(`zone_map.json`)

- `biome_map.json` と**同一グリッド・同一フォーマット**(固定3文字コードの `rows` 連結+`legend`)。別ファイルとして出力する。
- **海セルの自動割当**: 書き出し時に、水域セルへ**最も近い塗り済み陸セルの地理圏**を自動で書き込む(BFS/距離変換。等距離はセル座標の辞書順で決定論に)。→ 人間は海を一切塗らなくてよい。
- `legend` には各圏の `code / name / jp / color` を含める。

## 5. 実装メモ(既存機構の流用)

- グリッド: `state.grid` と同構造の `state.zoneGrid` を追加(保存・読込も同型)。
- モード切替: 「レイヤー1(バイオーム)/レイヤー2(地理圏)」のトグル。レイヤー2 では描画先とパレット定義を差し替えるだけで、ペイント・塗りつぶし・PNG入出力のコードパスは共通化できる想定。
- 水域判定はレイヤー1 を参照(リアルタイム。レイヤー1 で海岸線を直した場合はレイヤー2 の該当セルを未塗りに戻す)。
- 読込互換: `zone_map.json` が無い旧プロジェクトはレイヤー2 を空で開く。
